# Product Requirements Document (PRD)

## Project Overview
Brainzy - AI Study Companion is an AI-powered academic learning companion that helps students study effectively through automated syllabus grounding, interactive tutoring, study planning, progress tracking, and dynamic quiz generation.

## Features
1. Academic Storage: Manage syllabus materials (PDFs, DOCX, TXT, MD).
2. Doubt Bot: Smart tutoring assistant.
3. AI Summary Synthesis: Generate summaries from academic papers.
4. Active Recall Flashcards: Review terms, flip cards to show solutions, rate difficulties.
5. Practice Exam: Customize question counts, solve MCQs, and get instant grading feedback.
6. Study Timer: Run Pomodoro focus intervals.
7. Progress Tracker: Log manual study sessions.
8. Roadmap & Planner: Schedule active revision itinerary slots.

## Technology Stack
- Frontend: Next.js 16, React 19, TypeScript, TailwindCSS
- Backend: Lemma SDK
