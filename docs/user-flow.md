# User Flow

## Student Journey

### 1. User Opens Application

* Student lands on the home page.
* Student signs in or creates an account.

### 2. Dashboard

The dashboard displays:

* Daily study goals
* Learning streak
* Progress summary
* Recommended tasks

### 3. Choose Learning Activity

The student can:

#### AI Tutor

* Ask questions
* Receive explanations
* Get examples and guidance

#### Study Planner

* Set study goals
* Generate personalized schedules
* Track completed tasks

#### Quiz Generator

* Select a topic
* Generate a quiz
* View score and feedback

### 4. Progress Tracking

* Study sessions are recorded.
* Quiz scores are stored.
* Learning progress is updated.

### 5. Personalized Recommendations

Based on performance:

* Weak topics are identified.
* Revision suggestions are generated.
* New learning goals are recommended.
