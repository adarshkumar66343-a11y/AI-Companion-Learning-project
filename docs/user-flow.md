# User Flow

## Student Journey

### 1. Land & Select Activity
- The user opens the homepage displaying the 9 core productivity tool cards.
- Clicking any card redirects the user directly into the **Workspace Sandbox** (`/app`), focused on their chosen tab.

---

### 2. Workspace Navigation (Tabs)
The student navigates between tabs in the workspace header:
- **AI Summary**: Check paper inputs, click Synthesize, and read summaries.
- **Doubt Bot**: Chat with the grounded AI assistant.
- **Flashcards**: Review recall cards, flip, and rate difficulties.
- **Practice Exam**: Set question count limits, select MCQ choices, and view graded scores.
- **Study Timer**: Preset Pomodoro focus blocks and track intervals.
- **Progress Tracker**: View streaks, analyze weekly distributions, and log new study logs on the sidebar.
- **Roadmap**: Generate curriculum pathways based on grounded papers.
- **Planner**: Manage schedules, toggle completed tasks, and delete revision entries.
- **Storage**: Upload PDF files directly to the pod `/knowledge` folder and manage active paper contexts in the library.


---

### 3. Theme Toggle
- The user toggles the Light/Dark mode header button at any time to shift colors to the deep-slate theme.
